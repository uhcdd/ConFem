11.3 Nodal Output

Nodal output values are written to the file <name>.nodeout.txt for time intervals as prescribed in the step-section with the sub-keyword *NODE FILE, see Section 9.9.

An output time instant is indicated by a marker Time and the respective time value in the same line separated by a comma. This is followed by data at nodes, each line-wise with comma delimiters.

This includes the following nodal data

  • node number

  • coordinates

  • displacements corresponding to nodal degrees of freedom, e.g. in case of plates, in case of 2D-beams, in case of shells

    • i.e. the number of values varies according to the type of freedom, see also Section 1

  • internal nodal forces corresponding to nodal degrees of freedom, i.e. the same number of values as for displacements

    • internal nodal forces with absolute values smaller than roughly (depends on prescribed equilibrium tolerance , see Section ) may be considered as equilibrated

    • larger values (absolute) occur with prescribed degrees of freedom and correspond to boundary reactions

  • output for nodes of slab elements (SB3) partially differs
    integration point moments projected to nodes are inserted between displacements and internal nodal forces
