8.2 Using the .EXE-Version

https://concrete-fem.com

Python scripts may be compiled into a <name>.exe to be run under Windows. This requires the PyInstaller-package (https://pyinstaller.org/en/stable/). For compilation of ConFem python scripts go to the source file directory, open a powershell or command prompt window and launch 

PyInstaller .\ConFemAll.py

ConFemAll.py offers two hard coded versions:

  • A batch processing version for continuous feeding of input files listed in an arbitrary text file, see https://github.com/uhcdd/ConFem/tree/master/WinBatch for more instructions.

  • A prompt version as a wrapper for all modules, see https://github.com/uhcdd/ConFem/tree/master/WinPrompt for more instructions.

After successful compilation the executable is available under the dist-directory and my be started immediately without further installation requirements. It is basically a portable and may be copied – including the _internal-directory – to other windows sites and used there1 without a python installation.
