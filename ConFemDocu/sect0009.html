11.4 Stability_Tolerance_Factor

The option has a data line

stability_tolerance_factor,


with

allows for an -times deviation of the latest reaction (absolute value) in relation to the quadratic extrapolation of the three reactions (absolute values) immediately before.
 

The computation is terminated if this option is defined and the condition is violated.



A value in the range might make sense for quasistatic simulations. In case it is defined it should generally be larger for dynamic calculations with undamped oscillations.
