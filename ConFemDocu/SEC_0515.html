10.6 Beam Section

https:://concrete-fem.com

A section starting with *BEAM SECTION defines a set of beam elements (B23E,B23,BAX23,B21,B21E,BAX21E + embedded counterparts), whereby all elements within a set have the same properties.
Notice: beam elements may include a reinforcement. Their embedded counterparts ARE a reinforcement within an underlying 2D or axisymmetric continuum.

The section keyword *BEAM SECTION has as options

ELSET=<name>

refers to a name of a set of elements (mandatory)

 

(the same name also has to be defined within an element section)

MATERIAL=<name>

refers to a material name defining the material of the elements (mandatory)

 

(the same name also has to be defined within a material section)

SECTION=RECT

rectangular cross section

 

or

SECTION=POLYLINE

cross section shape described by a polyline

 

or

SECTION=CIRCLE

circular cross section

 

one of the preceding three is mandatory
 

SECTION=CIRCLE is mandatory for embedded beam versions of beam elements B23,B23E,BAX21E

REINF=<name>

refers to a reinforcement name defining the reinforcement of the beam elements (optional)
 

(the same name also has to be defined within a reinforcement section)

NREBAR=<float>

connected to a set of embedded elements of type B23I,B23EI,BAX21EI (optional), is ignored for other element types

 

works for CIRCLE-sections only

 

specifies the number of rebars per unit width – with length unit as is used in data set – to consider bonding

 

if not specified default is is with thickness of underlying CP*,CAX*-elements

 

explicit definition of NREBAR is recommended for BAX21EI-elements
