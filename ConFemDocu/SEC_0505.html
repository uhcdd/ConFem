10.8 Beam reinforcement

https:://concrete-fem.com

A section starting with *BEAM REINF defines the geometry of reinforcement layers. It is used in connection with a beam section.

The section keyword *BEAM REINF has as option

NAME=<name>

specifies a unique name which basically may be freely chosen



The following data lines describe a reinforcement layer with the following geometric properties

effective reinforcement ratio (used only for tension stiffening)

bond parameter (: no tension stiffening)



The reinforcement material properties are defined with the material type *RCBEAM. Further data lines may follow for further reinforcement layers if required. The number of reinforcement layers is not restricted.

The section keyword *BEAM REINF has no own sub-keywords.
