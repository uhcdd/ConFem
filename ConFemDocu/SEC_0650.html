8.1 Using Python Scripts

https://concrete-fem.com

For ConFem.py edit ConFemData.py and for ConSimFem.py or ConPlaD.py or ConSimplex.py or ConPostPlot.py edit the final section 

if __name__ == "__main__":
    ...
    Name="../DataExamples/E2/E2-02C1"               # input data name
    ...

to provide a particular input data set following the variable Name. The respective module (i.e. ConFem.py not ConFemData.py) is started by an IDE, see Section 1, or by invoking the Python-interpreter (version 3.11) directly – supplied with the respective module name, e.g. python ConFem.py – from a prompt window, e.g. windows powershell.
