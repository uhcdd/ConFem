9.2 Node Section

https:://concrete-fem.com

The section keyword *NODE has no options and no own sub-keywords. The data lines generally describe the following nodal data

<node number>,

<1st coordinate>,

<2nd coordinate> ,

<3rd coordinate>



Nodal data lines have to be extended in case of shell elements by

<node number>,

<1st coordinate>,

<2nd coordinate> ,

<3rd coordinate>,

<1st director coordinate>,

<2nd director coordinate> ,

<3rd director coordinate>



Values have to be separated by commas.

  • Nodal data should precede all other data.
