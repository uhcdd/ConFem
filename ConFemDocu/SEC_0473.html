11 Options File

https://concrete-fem.com

The options file (<name>.opt.txt) can be used to request specific outputs. There are three kinds of output that can be requested, depending on the type of analysis:

  • writeNodes - output nodal data depending on time or displacements

  • LineSearch - line search iteration method

  • reinforcementDesign - for use with ConPlad

  • max_equi_iterations - to control the number of iteration attempts before terminaing

These optional requests are included in one options file. Lower or upper capitals may be used.

  • 11.2 LineSearch
