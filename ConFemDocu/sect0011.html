ConFem Documentation : Organization of ConFem Output Data : Overview

12.1 Overview

Files for results are organized as follows.

Files with a type protocol report about every iteration of every instant of time of every simulation step.

Files with a type nodeout have output sections for instants of time as defined by the step sub-keyword *NODE FILE, see Section 10.9. Every node has a line within every output section with data for coordinates, values for nodal degrees of freedom and internal nodal forces. Types of nodal degrees of freedom an nodal forces depend on the element type, i.e. are different for, e.g., beam elements and plate elements.

Files with a type elemout also have output sections for instants of time as defined by the step sub-keyword *EL FILE, see Section 10.9. Output sub-sections correspond to every element. Every element integration point has a line within every element output sub-subsection with data for coordinates and further values like, e.g. generalized strains and stresses and further values depending on the element and material type.

Files with a type timeout have a line for all computed instants of time with values of every nodal degree of freedom as is defined in a file of type opt, see Section 11.1.
